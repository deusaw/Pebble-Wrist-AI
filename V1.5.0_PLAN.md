## 当前不可破坏的 TTS 基线

- Emery `TTS_RING_SIZE = 35840`，运行时堆分配
- Flint `TTS_RING_SIZE = 12288`，运行时堆分配
- `TTS_DECODE_BYTES = 1600`
- `TTS_MIN_WRITE_BYTES = 800`
- `TTS_PLAYBACK_MS = 100`
- Emery `TTS_START / RESTART = 24000 / 8000`
- Emery `TTS_PAUSE / RESUME = 30500 / 24500`
- Flint `TTS_START / RESTART = 7500 / 6000`
- Flint `TTS_PAUSE / RESUME = 8500 / 6500`
- `TTS_CHUNK_SIZE = 350`
- `TTS_SEND_DELAY_MS = 43`
- `TTS_STREAM_MAX_RETRIES = 80`
- pending write 成功 flush 后，同轮必须继续 decode/write。
- 不得恢复 8-bit PCM 数字缩放音量。
- TTS 播放期间继续禁用 ScrollLayer 动画。
