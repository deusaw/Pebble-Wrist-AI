# Wrist AI v1.5.0 Release Notes

Upgrade path: v1.4.0 → v1.5.0

## Dashboard copy · English

Wrist AI 1.5 turns your Pebble into a far deeper personal assistant.

Talk naturally to create reminders, Timeline events and multiple ToDos. Keep
them organized in ToDo & Notes, sync them with Todoist, and return to the
conversation that created them. Wrist AI can now remember useful long-term
context, understand exact sleep sessions and multi-day health trends, and offer
practical health suggestions instead of only repeating numbers.

The watch experience has also been rebuilt: configurable themes, clearer
Pebble-style menus, a re-record flow for Dictation, an optional on-watch
Controls tutorial, better Chinese handling, and safer navigation.

On Pebble Time 2, TTS now supports high-quality 16 kHz playback with improved
clarity, smoother streaming and safer volume control. Experimental Pebble 2 Duo
TTS is also included.

This is the largest Wrist AI update yet.

## Complete changes · English

### Health intelligence

- Adds optional 1, 3 or 7-day health context covering steps, activity,
  distance, calories, heart rate, sleep and deep sleep.
- Sends exact sleep and deep-sleep intervals, not aggregate minutes alone.
- Correctly treats the latest noon-to-noon sleep record as the latest sleep,
  while avoiding invalid comparisons between incomplete current-day activity
  and complete historical days.
- Health prompts now ask for trends, anomalies and practical suggestions rather
  than a raw-number recap.

### Voice, language and replies

- Optional launch Dictation for deliberate Quick Launch and Timeline-action
  entry points.
- Adds a Dictation review page: SELECT sends, DOWN re-records and BACK cancels.
- Automatically retries once when a system notification interrupts Dictation.
- Normalizes supported Traditional Chinese Dictation/model text to Simplified
  Chinese before titles, display, storage, Timeline parsing and TTS.
- Limits watch-oriented model replies and strips links, source URLs and web
  citations that are not useful on a watch or in spoken output.
- Reworks Config conversation export using explicit UTF-8 bytes, BOM and an
  old-WebView fallback.

### Long-term Memory

- Adds a private phone-side `Memory.md`.
- The model may add, update and remove controlled memory entries containing
  stable preferences and useful personal context.
- Explicit stable facts such as “remember my birth date” are routed exclusively
  to Memory and cannot be misclassified as a Note/TODO.
- A dedicated Memory planner and deterministic birth-date fallback recover when
  the main model omits or damages its hidden memory command.
- Config can display and edit the complete Markdown document with format,
  sensitive-data, item-count and byte-size validation.
- Secrets, credentials and exact health samples are excluded.

### Timeline, reminders and launch actions

- Creates local Pebble Timeline pins through natural language.
- Understands ordinary phrases such as “remind me in five minutes” without
  requiring the user to say “Timeline”.
- Supports relative times, exact times, date-only all-day events and up to five
  events in one request.
- Distinguishes a timed reminder from a true all-day event.
- Wrist AI-created pins can be updated or deleted from linked ToDo & Notes.
- Adds optional strong local reminders using persistent watch Wakeups and a
  longer vibration pattern.
- Timeline actions can reopen Wrist AI with launch context and restore the
  originating conversation through its saved `chat_id`.

### ToDo & Notes

- Adds a watch-native ToDo & Notes area for untimed tasks, notes and linked
  Timeline events.
- A single request can create up to five items.
- Items retain their originating conversation and can reopen it from the watch.
- Supports Complete, Reopen and Delete with durable deferred synchronization.
- Completed items move to a separate Archive entrance; the latest 50 are kept.
- Timeline rows are clearly labelled and show local time or `All day`.
- Config can edit titles/content, complete, reopen and delete local items.

### Todoist two-way sync

- Adds optional direct Todoist synchronization without a Wrist AI server.
- Syncs untimed ToDos and timed Timeline-linked events.
- Imports Todoist creation, edits, completion, reopening, due-time changes and
  deletion back into Wrist AI.
- Uses stable task mappings, incremental sync tokens and a durable offline
  mutation queue.
- Syncs on launch, after local changes and every five minutes while Wrist AI
  remains open.
- Optional 30-minute, hourly or three-hour watch Wakeups provide background
  synchronization.
- `Syncing Todoist`, `Synced Todoist`, offline and failure states appear below
  the conversation title. If sync is enabled without a token, the watch shows
  `Set up Todoist`; disabled users see no prompt.

### TTS and speaker watches

- Adds conservative experimental TTS support for Pebble 2 Duo.
- Pebble Time 2 high-quality mode uses 16 kHz μ-law transport decoded to native
  16-bit speaker output.
- Adds DC removal, a smoothed noise gate, envelope compression, gentle
  high-frequency smoothing, peak limiting and sentence-boundary de-clicking.
- Larger AppMessage packets and guarded flow control improve playback when the
  iPhone Pebble app is backgrounded.
- Keeps the watch menu at 10–100% while capping hardware output at 50% to reduce
  high-volume clipping.
- Config can select high-quality or standard audio and set speaking rate.
- SELECT stops playback; scrolling remains available without competing
  animations.

### Watch UI and Config

- Adds Random, Orange, Violet, Indigo, Cobalt, Oxford Blue, Red and Deep Gray
  themes.
- Preserves a light reply page while using compact themed headers and
  white-on-theme menu selection for reliable contrast.
- Improves Chinese reply sizing and header alignment.
- Adds an on-watch Controls tutorial for short/long button presses. It can be
  removed from the UP menu in Config after onboarding.
- DOWN opens ToDo & Notes; UP keeps conversations, models, Web Search, volume
  and settings together.
- BACK from a conversation opened through the UP menu returns to that menu.
- Config is reorganized into clearer sections with Wrist AI orange/purple
  styling and bounded data payloads.

### Reliability and compatibility

- Keeps the existing UUID `1c4a7728-ee62-431f-a941-e40fafab98af`.
- Preserves the single-file PebbleKit JS bundle required by iOS Core Devices.
- Fixes watch crashes around Note Complete/Reopen/Delete, Archive reloads,
  startup sync, stale commands and unsafe decimal parsing on Flint.
- Uses independent limits for active items and the 50-item Archive.
- Builds for Basalt, Chalk, Diorite, Emery and Flint.

## 完整更新 · 中文

### 健康智能

- 新增可选的 1、3、7 日健康上下文，包含步数、活动、距离、热量、心率、睡眠和深睡。
- 向模型提供精确的睡眠与深睡起止区间，不再只有汇总分钟数。
- 正确识别最近一次中午到中午的睡眠记录，同时避免把不完整的当天活动与完整历史日直接比较。
- 健康分析会总结趋势、异常并给出可执行建议，而不是只复述数字。

### 语音、中文与回复

- Quick Launch 与 Timeline Action 可按 Config 设置启动录音。
- 新增语音确认页：SELECT 发送、DOWN 重录、BACK 取消。
- 系统通知打断录音时自动重试一次。
- 在标题、显示、存储、Timeline 解析和 TTS 前，将支持的繁体语音/模型文本转为简体。
- 限制手表回复长度，并移除对手表与朗读无用的链接、来源 URL 和网页引用。
- Config 对话导出改为明确的 UTF-8 字节、BOM 与旧 WebView 兼容回退。

### 长期记忆

- 新增保存在手机端的隐藏 `Memory.md`。
- 模型可受控地新增、修改和删除稳定偏好与长期有用的个人上下文。
- “记住我的出生日期”等稳定个人事实只会进入长期记忆，不再误建为 Note/TODO。
- 主模型漏掉或损坏隐藏记忆命令时，由专用 Memory Planner 与生日日期本地兜底恢复。
- Config 可查看和编辑完整 Markdown，并检查格式、敏感信息、条数与字节上限。
- 不允许记忆密钥、凭证和精确健康样本。

### Timeline、提醒与启动联动

- 通过自然语言创建本地 Pebble Timeline Pin。
- “五分钟后提醒我”这类普通表达无需明确说出 Timeline。
- 支持相对时间、精确时间、日期型全天事件，以及单次最多五个事件。
- 区分定时提醒和真正的 All Day Event。
- Wrist AI 创建的 Pin 可通过关联的 ToDo & Notes 更新或删除。
- 明确要求强提醒时，可使用持久化 Wakeup 与更长震动。
- Timeline Action 打开 Wrist AI 时携带事件上下文，并通过 `chat_id` 恢复原始对话。

### ToDo & Notes

- 新增符合手表操作的 ToDo & Notes，用于无时间待办、笔记与 Timeline 关联事件。
- 单次请求最多创建五项。
- 每项保存创建它的对话，可从手表返回原对话。
- Complete、Reopen、Delete 使用持久化延迟同步，降低 UI 回调崩溃风险。
- 已完成事项进入独立 Archive 入口，保留最新 50 条。
- Timeline 条目有明确标识，并显示本地时间或 `All day`。
- Config 可编辑标题/内容、完成、重开和删除事项。

### Todoist 双向同步

- 无需 Wrist AI 自建服务器，直接连接 Todoist。
- 同步无时间 ToDo 与带时间的 Timeline 关联事件。
- Todoist 中的新建、编辑、完成、重开、日期变化和删除可回写 Wrist AI。
- 使用稳定任务映射、增量同步 Token 与持久化离线操作队列。
- 启动时、本地变更后，以及 Wrist AI 保持打开期间每五分钟同步。
- 可选 30 分钟、1 小时或 3 小时 Wakeup 后台同步。
- `Syncing Todoist`、`Synced Todoist`、离线和失败状态显示在对话标题下方。
  启用但缺少 Token 时显示 `Set up Todoist`；未启用的用户不会看到提示。

### TTS 与扬声器设备

- 新增保守的 Pebble 2 Duo 实验性 TTS。
- Pebble Time 2 高质量模式使用 16 kHz μ-law 传输并解码为原生 16-bit 播放。
- 加入直流去除、平滑噪声门、包络压缩、轻度高频平滑、峰值限制和句间去爆点。
- 使用更大的 AppMessage 音频包和保护性流控，改善 iPhone Pebble App 在后台时的播放。
- 手表菜单仍为 10–100%，底层硬件上限为 50%，降低高音量削波。
- Config 可切换高质量/标准模式并调整语速。
- SELECT 可停止朗读；播放期间仍能滚动且不会叠加滚动动画。

### 手表 UI 与 Config

- 新增 Random、Orange、Violet、Indigo、Cobalt、Oxford Blue、Red、Deep Gray 主题。
- 回答页保持浅色；菜单使用紧凑主题标题栏和白色选中文字，保证可读性。
- 改进中文回复字号与标题垂直位置。
- 新增短按/长按 Controls 新手教程；熟悉后可在 Config 中从 UP 菜单关闭。
- DOWN 打开 ToDo & Notes；UP 集中放置对话、模型、Web Search、音量与设置。
- 从 UP 菜单进入对话后，BACK 会返回 UP 菜单。
- Config 重新分组并采用 Wrist AI 橙紫风格，同时限制数据载荷大小。

### 稳定性与兼容性

- 保持原 UUID `1c4a7728-ee62-431f-a941-e40fafab98af`。
- 保留 iOS Core Devices 所需的单文件 PebbleKit JS 封包。
- 修复 Note 完成/重开/删除、Archive 刷新、启动同步、陈旧命令和 Flint 数字解析相关崩溃。
- Active 项与 50 条 Archive 使用独立容量限制。
- 支持 Basalt、Chalk、Diorite、Emery 和 Flint。
